# flowci-plugin-docker-build
build and push docker image

#### master branch ####
- tag: `master.{build number}`
- ex: `master.15`

#### develop branch ####
- tag: `develop.{build number}`
- ex: `develop.15`

#### feature branch ####
- tag: `feature.{feature id}.{build number}`
- ex: `feature.322.15`
